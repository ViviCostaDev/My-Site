MINHAS LINGUAGENS


De acordo com o Wikipédia, o HTML5 (Hypertext Markup Language) é uma linguagem de marcação de hipertexto para World Wide Web e é uma tecnologia chave da internet originalmente proposto por Opera Software. Esta é a quinta versão da linguagem HTML. A nova versão traz consigo importantes mudanças quanto ao papel da linguagem no mundo da web através de novas funcionalidades como, semântica e acessibilidade. Possibilitando assim, o uso de novos recursos ao melhorar a linguagem com o suporte para as mais recentes multimídias. O HTML é a estrutura das páginas web.

O Javascript é uma linguagem de programação estruturada, de script de alto nível e tipagem dinâmica fraca. Orientada a objetos, a linguagem de programação Javascript em conjunto com o HTML e o CSS formam uma das três principais tecnologias da World Wide Web. Essa tecnologia criada em 1995 por Brendan Eich para o navegador Netscape Navigator, revolucionou o mundo web ao permitir páginas interativas. Atualmente é a principal linguagem para programação client-side em navegadores web. Além de ser uma linguagem de programação front, o Javascript também pode ser no back com tecnologias como o Node.js.

O CSS (Cascading Style Sheets) nasceu em 1994 para facilitar a programação web quando trouxe consigo mecanismos para adicionar estilos às páginas. As folhas de estilos podem ser aplicados diretamente dentro do próprio HTML ou em arquivo externo através de link. Com ele é possível dar movimentos a elementos de uma página, mover itens, alterar cores, altura, dentre outras coisas. No entanto, a interpretação do CSS varia de navegador para navegador, e essa interpretação pode ser valiada com o teste Acid2, que é uma página de teste de compatibilidade com padrões web W3C com navegadores publicada pela Web Standards.
